# Task Overview

The goal of this project is to develop a basic yet powerful to-do application that allows users to:

Add and organize tasks

Set date and time for tasks

Edit existing tasks

Mark tasks as completed

Delete tasks

Retain tasks even after page refresh using localStorage

This project focuses on functionality, usability, and modern UI design.

# Features

Add new tasks with date & time

Mark tasks as completed

Edit tasks

Delete tasks

Persistent data storage using browser localStorage

Fascinating UI/UX with gradients and smooth animations

Responsive design for all screen sizes

# Technologies Used

HTML5 – Structure of the application

CSS3 – Styling, animations, and modern UI

JavaScript (ES6) – Application logic and interactivity

Project Structure
├── index.html      # Main HTML file
├── style.css       # CSS styling
├── script.js       # JavaScript logic
└── README.md       # Project documentation

Open the project folder

Double-click on index.html
OR

Open index.html using any modern web browser

# Expected Output

A visually attractive to-do list interface

Users can add tasks with date & time

Tasks can be marked completed, edited, or deleted

Tasks remain saved even after refreshing the page

# Learning Outcomes

Understanding DOM manipulation

Working with browser localStorage

Implementing CRUD operations in JavaScript

Designing modern UI using CSS

Improving frontend development skills
